# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-05-08

### Added
- Initial release of Kiro Dock with advanced features
- Configurable dock positioning (bottom, top, center, top-left, top-right)
- Setup wizard for easy configuration
- Support for custom animations and icon assets
- Adjustable animation speed (slow, normal, fast)
- Configuration file storage at ~/.config/kiro-dock/config.json
- Kiro CLI integration with real-time chat streaming
- Tray icon synchronization with animation
- Draggable agent and chat popup windows
- Auto-start capability
- Comprehensive documentation and troubleshooting guide

### Features
- [x] Animated desktop agent
- [x] Multiple position options
- [x] Custom asset support
- [x] Configuration wizard
- [x] Chat integration
- [x] Tray icon
- [x] Keyboard shortcuts
- [x] Auto-login support

### Security
